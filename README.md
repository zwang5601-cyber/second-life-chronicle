# 异世界人生录 (Second Life Chronicle)

![Godot 4.3](https://img.shields.io/badge/Godot-4.3-blue)
![License](https://img.shields.io/badge/License-MIT-green)

> 回合制RPG + 据点经营 + 家园防护

## 🎮 游戏特色

- ⚔️ **八方旅人式战斗** - 破盾系统 + BP爆发 + 弱点克制
- 🏠 **据点经营** - 建筑升级、锻造、炼金、商店、农田
- 🛡️ **家园防护** - 夜袭防守战、防御设施建造
- 📈 **角色成长** - 等级、职业、技能树、装备系统
- 📖 **异世界转生** - 沉浸式剧情体验

## 🛠️ 技术栈

- **引擎**: Godot 4.3
- **语言**: GDScript
- **画面**: 高精像素风 + 现代光影
- **形式**: 混合模式（据点探索 + 节点式冒险）

## 📁 项目结构

```
second-life-chronicle/
├── assets/          # 资源文件
│   ├── images/      # 图片资源
│   ├── audio/       # 音频资源
│   └── data/        # 数据配置
├── src/             # 源代码
│   ├── autoload/    # 全局单例
│   ├── battle/      # 战斗系统
│   ├── entities/    # 实体类
│   ├── systems/     # 游戏系统
│   ├── management/  # 经营系统
│   ├── defense/     # 防护系统
│   └── dialogue/    # 对话系统
├── scenes/          # 场景文件
└── docs/            # 文档
```
